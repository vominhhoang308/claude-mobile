# Feature Specs

This directory holds per-feature specification files generated by the
`/speckit.specify` workflow.

Each feature goes through three artifacts before implementation:

| File | Generated by | Purpose |
|------|-------------|---------|
| `spec.md` | `/speckit.specify` | User stories, acceptance criteria, out-of-scope items |
| `plan.md` | `/speckit.plan` | Architecture decisions, file changes, complexity tracking |
| `tasks.md` | `/speckit.tasks` | Ordered, testable implementation tasks |

## Usage

```bash
# 1. Write a spec for a feature
/speckit.specify

# 2. Design the implementation plan
/speckit.plan

# 3. Generate tasks
/speckit.tasks

# 4. Implement
/speckit.implement
```
